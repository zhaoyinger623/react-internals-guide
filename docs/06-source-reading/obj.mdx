[Render·Hooks](/02-update-pipeline/t7)

[Render·Hooks](/02-update-pipeline/t7)/[Render·变更标记](/02-update-pipeline/t10)

位运算:`& | -lanes`

[Schedule·Lane](/02-update-pipeline/t13)

**对象之间的关系：**JSX 编译成 `ReactElement` → 更新生成带 `Lane` 的 `Update`，并进入 `Fiber` 的 `UpdateQueue` → Scheduler 安排任务 → reconciler 遍历 Fiber、处理 Hook 并生成带 `flags` 的树 → commit 提交变更。
