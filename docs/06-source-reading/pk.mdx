包

角色

关键内容

平台相关?

`react`

核心 API

`createElement`、`Component`、`Hooks` 的「壳」、`createContext`

❌ 无关

`react-reconciler`

协调器(WHAT)

Fiber、Diff、beginWork/completeWork、commit、Hooks 真正实现

❌ 无关

`react-dom`

渲染器 + 入口(HOW)

`createRoot`、事件系统、**HostConfig**(DOM 操作实现)

✅ 相关

`scheduler`

调度器(WHEN)

时间切片、优先级、MessageChannel、最小堆

❌ 无关

**HostConfig 用于隔离平台差异。**`react-reconciler` 通过 `createInstance`、`appendChild`、`commitUpdate` 等接口操作宿主节点。`react-dom` 提供 DOM 实现，React Native 的渲染器提供原生视图实现，因此 reconciler 不需要包含具体平台的节点操作。
