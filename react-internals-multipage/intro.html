教程以一次 setState 更新为主线。每章对应更新流程中的一个环节，章节顶部会标出当前位置。Fiber、Hook、Lane 和 Scheduler 会放回同一条调用链中讲解。

本教程从第一层开始：先理解 React 平时如何渲染组件、更新界面；再逐步进入 Fiber、Diff、Hooks、调度与并发。每一节都有讲解、图示和精简源码，帮助你把一个概念看具体。

建议顺序：先读「一次 setState 的旅程」了解整体流程，再按第一层到第四层往下读。源码术语会在对应环节中解释；文末的源码地图可用于查找包、文件和核心对象。

三个主要模块

一次更新会依次涉及 Scheduler、Reconciler 和 Renderer。后续章节会分别展开它们的职责和协作方式。

🔵 小圆点 = 一次 update,依次流过三大模块最终落到屏幕(循环播放)

Scheduler=何时做(WHEN) · Reconciler=做什么(WHAT,算出变更)· Renderer=怎么落地(HOW,改真实 DOM)。
