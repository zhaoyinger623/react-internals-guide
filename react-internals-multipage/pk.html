A1 · 宏观包结构

动手看源码前,先建立地图。React 是一个 monorepo,packages/ 下有 30+ 个包。做 Web 开发,真正要理解的只有 4 个核心包,它们正好对应前面反复出现的三大模块:

包角色关键内容平台相关?
react核心 APIcreateElement、Component、Hooks 的「壳」、createContext❌ 无关
react-reconciler协调器(WHAT)Fiber、Diff、beginWork/completeWork、commit、Hooks 真正实现❌ 无关
react-dom渲染器 + 入口(HOW)createRoot、事件系统、HostConfig(DOM 操作实现)✅ 相关
scheduler调度器(WHEN)时间切片、优先级、MessageChannel、最小堆❌ 无关
HostConfig 用于隔离平台差异。react-reconciler 通过 createInstance、appendChild、commitUpdate 等接口操作宿主节点。react-dom 提供 DOM 实现，React Native 的渲染器提供原生视图实现，因此 reconciler 不需要包含具体平台的节点操作。
